## Developer information

### How to run rspec test
Start spork:
{{{
spork
}}}
Open other shell and type:
{{{
rspec .
}}}

### How to run cucumber tests
Start spork:
{{{
sprok cucumber
}}}
Open other shell and type:
{{{
cucumber -p all
}}}
which means run all the tests, or
{{{
cucumber
}}}
to run only the current test (if any)
